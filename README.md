## installation
```shell
npm install @jswork/package-cli
```

## usage
~~~
Usage: pkg [options]

Options:
  -V, --version      output the version number
  -n, --npm-install  Get npm install script.
  -s, --shortname    Get short name.
  -h, --help         display help for command
~~~
